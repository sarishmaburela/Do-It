# Do It - Task Manager

A simple and responsive task manager built using HTML, CSS, and JavaScript.

This app allows users to add, complete, filter, and delete tasks with progress tracking.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter by All, Active, and Done
- Progress bar
- Local storage support
- Responsive design

## Built With

- HTML
- CSS
- JavaScript

- `index.html` – Main application file
